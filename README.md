# Todo List

A web application that allows users to add, manage, and track their daily tasks.  
Users can mark tasks as completed and remove individual or all todos.

## Features
- Add a new todo
- Mark todos as completed
- Delete a single todo
- Delete all todos
- Count total todos

## Technologies Used
- React
- CSS

## Purpose
This project was created to practice React fundamentals and build a simple task management application.
